# try-render-model-web-gpu

Quick scratch project for learning.

## Style
- Ignore the project name.
- Prioritize concise and simple files with few dependencies.
- Prioritize quick setup, with few steps.
- Use python for main project logic unless there is a compelling reason to mix in a different language.
- All computation will be local, with no traditional backend.

- Render a blue sphere on a black background using three.js
- Visualize it in the browser
